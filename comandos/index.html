• Diversão
╰ K!Pikachu - Gera uma imagen aleátoria do pikachu.
╰ K!Wasted - Gera uma imagen de prisão do GTA V.
╰ K!Triggered - Provoque o usuário mencionado/autor.
• Configurações
╰ K!Config - Faça alterações nas configurações do meu sistemas em seu servidor.
• Informações
╰ K!Avatar - Pegar avatar do usuário usando ID ou Menção.
╰ K!BotInfo - Veja minhas informações.
╰ K!ChannelInfo - Veja informações de um canal de texto/voz usando o ID ou Menção.
╰ K!Convite - Link de convite para me adicionar em seu servidor.
╰ K!Ping - Mostra a Latência da API do Discord e a da Maquina de Hospedagem.
╰ K!Serverinfo - Pega informações do servidor onde o comando foi executado.
╰ K!UserInfo - Pega informações do usuário usando ID ou Menção.
╰ K!Stats - Veja informações basicas sobre min.
╰ K!Warns - Veja se você ou o usuário menciona tem avisos.
• Administração
╰ K!Clear - Limpe mensagens do canal onde o comando foi executado com o limite de 100 mensagens.
╰ K!Lock - Trave o canal onde o comando foi executado.
╰ K!Unlock - Link de convite para me adicionar em seu servidor.
╰ K!Ban - Mostra a Latência da API do Discord e a da Maquina de Hospedagem.
╰ K!Unban - Pega informações do servidor onde o comando foi executado.
╰ K!Kick - Pega informações do usuário usando ID ou Menção.
╰ K!Remove-warn - Veja informações basicas sobre min.
╰ K!Warn - De uma aviso no usuário mencionado, se receber 3 aviso será banido permanentemente.
